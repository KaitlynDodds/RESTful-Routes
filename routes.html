RESTful Routes

Name Path Action Purpose
Index /books GET Show all of an entity
Show /books/:id GET Show a single instance of an entity
New /books/new GET Show the form to create a new entity
Create /books PUSH Create a new entity, redirect somewhere after entity has been created
Edit /books/:id/edit GET Show the form to edit an existing entity
Update /books/:id PUT Make changes to an existing entity, redirect after changes made
Delete /books/:id DELETE Delete an existing entity, redirect somewhere else
